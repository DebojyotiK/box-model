• Border Box: Height and width includes till border.For example if height,width=400px,padding=20px,border=5px then total size will be 400px and content will be 400-(20*2+5*2)=350px
  • Content Box: Height and width includes only content.For example if height,width=400px,padding=20px,border=5px then total size will be (400+20*2+5*2)px = 450px and content will be 400px
  • If height/width is given in percentage to the child then the size is determined on the content size of the parent
